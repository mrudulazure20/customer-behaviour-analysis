# Customer Shopping Behaviour Analysis

## Overview

The **Customer Shopping Behaviour Analysis** project focuses on analyzing customer purchasing patterns and shopping behavior using data analytics techniques. The project involves data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, interactive dashboard creation, and report generation to derive meaningful business insights.

The objective is to understand customer preferences, spending habits, and key factors influencing purchasing decisions, enabling data-driven business strategies.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer demographics
* Purchase history
* Product categories
* Transaction details
* Payment methods
* Shopping frequency
* Customer ratings and feedback

The dataset was imported into Python for preprocessing and analysis and later stored in PostgreSQL for advanced querying.

## Project Workflow

### 1. Data Loading

* Imported the dataset using Pandas.
* Verified data structure and column information.
* Checked data types and dataset dimensions.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected data inconsistencies.
* Standardized column formats.

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Examined purchasing trends.
* Identified popular product categories.
* Studied spending patterns and customer behavior.
* Generated visualizations to uncover insights.

### 4. PostgreSQL Analysis

* Created a PostgreSQL database.
* Loaded cleaned data into database tables.
* Executed SQL queries to answer business questions.
* Performed aggregations, filtering, and trend analysis.

### 5. Power BI Dashboard

* Connected Power BI to the processed dataset.
* Designed interactive dashboards.
* Created KPIs and visual reports.
* Enabled dynamic filtering and drill-down analysis.

### 6. Report Generation

* Summarized key findings.
* Documented customer behavior trends.
* Presented actionable business recommendations.

## How to Run the Project

### Prerequisites

Install the required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

### Steps

1. Download the dataset.
2. Open the Jupyter Notebook.
3. Load the dataset using Pandas.
4. Perform data cleaning and EDA.
5. Connect to PostgreSQL and load the cleaned data.
6. Execute SQL queries for analysis.
7. Import the processed data into Power BI.
8. Build and explore the dashboard.
9. Review the final report and insights.
